# taq-github-bot

> A GitHub App built with [Probot](https://github.com/probot/probot) that Taqtile&#x27;s Github bot

## Setup

```sh
# Install dependencies
yarn install

# Run typescript
yarn build

# Run the bot
yarn start
```
