# taq-github-bot

> A GitHub App built with [Probot](https://github.com/probot/probot) that Taqtile&#x27;s Github bot

## Setup

```sh
# Install dependencies
npm install

# Copy test.env to .env and change values according

# Run the bot
npm start

# Or run the tests
npm start
```
